 <p>    <h2>ABOUT PROJECT</h2>
        <p>Commander game, where user needs to place weapons, upgrade weapons on coins that he receives for killed enemies, and protect the territory</p>
        <h3>Project scalability</h3>
        <p>each level: position and proportions of shape, enemy health, level decorations is generated dynamically based on level number. Dynamic level generation allows me to create ability for user to create his own level and infinity level generation</p>
    </p>

 <p>
        <h3>Coins</h3>
        <p>user collects coins for completing levels</p>
  </p>
 <p>
        <h3>Integrations:</h3>
        <p>icloud</p>
       <p> progress is stored in icloud and coins for completed levels are stored in icloud keychain<br></p>
      </p>
